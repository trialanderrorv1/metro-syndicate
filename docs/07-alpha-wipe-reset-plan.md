# Alpha Wipe and Reset Plan

## Suggested policy
- closed alpha can wipe freely
- announce wipes ahead of time
- store a wipe reason and timestamp
- optionally preserve cosmetic names and forum accounts

## Recommended wipe moments
- before first external alpha
- after major economy rebalance
- after premium testing phase if real value was not involved

## Reset options
1. **Full wipe**
   - delete player progression, inventory, economy, forum threads
2. **Soft wipe**
   - keep account identity and forum profile
   - reset stats, items, currencies, jail/hospital, timers
3. **Economy-only wipe**
   - keep level/stats, reset cash, bank, inventory, market, premium coins

## Best alpha choice
- keep account identity
- reset gameplay state
- clearly label the season/build
