# Image-aligned shell redesign

## What changed
- rebuilt `client/App.tsx` around the generated mockup's structure
- left rail now uses a stacked browser-crime-game navigation layout
- center panel now defaults to a dedicated Crimes screen with large action rows and attempt buttons
- right rail is now a dense character/status column with profile, stat bars, and quick action buttons
- preserved the existing Metro Syndicate gameplay hooks and API calls while changing the presentation

## Design direction
- dark steel panels
- compact MMO/browser-game information density
- strong boxed sections and embossed buttons
- image-aligned shell without copying third-party branding or assets
